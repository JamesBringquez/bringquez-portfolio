# ARKĒ store assets

## Logo variants (`brand/`)

| File | Use |
|------|-----|
| `arke-logo-dark.png` | **Black garments only** — stealth tonal mark (dark gray on black + holo accents) |
| `arke-logo-light.png` | **White / pearl fabric** — high-contrast black letters + holo accents |
| `arke-brand-sheet.png` | Brand identity reference |

**Rule:** Never place the dark stealth logo directly on white or pearl fabric. Use the light mark on white and pearl pieces instead.
